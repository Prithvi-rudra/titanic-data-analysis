Titanic survival analysis using Python, Pandas, Matplotlib, and Seaborn

# 🚢 Titanic Data Analysis

## 📌 Project Overview

This project analyzes the Titanic dataset to understand survival patterns based on gender, passenger class, and age.

## 🛠 Tools Used

* Python
* Pandas
* Matplotlib
* Seaborn

## 📂 Dataset

* Titanic-Dataset.csv

## 🔍 Analysis Performed

* Data cleaning and preprocessing
* Missing value handling
* Survival analysis by gender
* Survival analysis by passenger class
* Age group analysis

## 📊 Visualizations

* Survival rate by gender
* Survival rate by passenger class
* Age distribution histogram

## 📈 Key Insights

* Female passengers had much higher survival rates than males
* First-class passengers had better survival chances
* Children had higher survival rates than older passengers
* Passenger class, gender, and age strongly influenced survival

## 🚀 Conclusion

The project illustrates the work of data analysis that has ability to reveal underlying structures and real-world understanding from retrospective datasets
